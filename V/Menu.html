Componentes pasivos de una red

RJ45

RJ45 es un conector que se utiliza para conectar a redes dispositivos mediante un cable que puede ser de hasta 8 hilos en su interior, el conector posee estos 8 pines, aunque no siempre se utilizan todos. El conector RJ45 puede ser macho o hembra, aunque lo más común es ver el macho en cables de red de ordenadores, también para conectar a una pared y hacer la conexión más cómoda se puede usar un conector hembra, estos conectores también están en switch de tipo doméstico o empresarial para conectar varios cables RJ45, el QNAP QGD-1600P analizado aquí posee 16 conexiones RJ45 para hasta 16 dispositivos.

Keystone

Un conector Keystone es un tipo de conector eléctrico utilizado para unir cables de telecomunicaciones en un panel de conexiones. Recibe su nombre de la piedra central de un arco, que es la piedra angular, que sostiene las otras piedras en su lugar. El conector de piedra angular también se conoce a veces como conector kwik.

Cable utp

El cable UTP, par trenzado sin blindaje pos sus siglas en inglés (UTP), es un cable balanceado sin blindaje con hilos de colores de par trenzado. Este tipo de cable está disponible en versiones de dos y cuatro pares, fue el tipo de cable dominante en el cableado de piso y cableado de terminales en la década de 1990 y forma parte del estándar de cableado 11801 y de las especificaciones EIA / TIA.

Patch panel

Un patch panel, también conocido como panel de conexión, es una pieza de montaje de hardware con varios puertos para conectar y gestionar los cables LAN o los cables de fibra/cobre entrantes y salientes.

Patch cord

Cable de conexión (patch cord) también llamado cable de red, se usa en redes de computadoras o sistemas informáticos o electrónicos para conectar un dispositivo electrónico con otro.Está compuesto por cobre y cubierto de plástico. Aunque esta definición se usa con mayor frecuencia en el campo de las redes informáticas, pueden existir cables de conexión también para otros tipos de comunicaciones electrónicas.

Fibra optica

La fibra óptica es un medio físico de transmisión de información, usual en redes de datos y telecomunicaciones, que consiste en un filamento delgado de vidrio o de plástico, a través del cual viajan pulsos de luz láser o led, en la cual se contienen los datos a transmitir.

Canaletas

Las canaletas para redes son elementos esenciales en la instalación de sistemas de comunicación y redes de datos. Existen diversos tipos de canaletas, cada una con características específicas que se adaptan a diferentes necesidades. En este artículo, te presentaremos los principales tipos de canaletas para redes.

Componentes activos de una red

Switch

Un switch es un dispositivo de red que se utiliza para conectar múltiples dispositivos dentro de una red local (LAN). Funciona en la capa 2 del modelo OSI, es decir, en la capa de enlace de datos, y utiliza direcciones MAC para dirigir el tráfico de red al dispositivo correcto.

Router

Un router es un dispositivo que se utiliza para dirigir el tráfico entre diferentes redes, generalmente entre una red local y una red externa, como Internet. Opera en la capa 3 del modelo OSI (capa de red) y utiliza direcciones IP para enrutar los paquetes de datos.

Tarjeta de red

Una tarjeta de red (NIC, Network Interface Card) es un componente hardware que se instala en un dispositivo, permitiendo que este se conecte a una red. Puede ser tanto para conexiones cableadas (Ethernet) como inalámbricas (WiFi).

Interruptor inteligente wifi

Un interruptor Smart wifi es aquel dispositivo capaz de conectarse a una red inalámbrica para controlar la iluminación, apagar y encender dispositivos, etc.

Acces point

Los AP o WAP (Access point o Wireless Access point) También conocidos como puntos de acceso. Son dispositivos para establecer una conexión inalámbrica entre equipos y pueden formar una red inalámbrica externa (local o internet) con la que interconectar dispositivos móviles o tarjetas de red inalámbricas.

Herramientas utilizadas para realizar cableado estructurado

Ponchadora

Una herramienta manual utilizada para insertar cables en conectores, como los conectores RJ45, que se usan comúnmente en redes de datos. La ponchadora asegura que los cables estén correctamente alineados y firmemente asegurados en los conectores, lo que es crucial para mantener la integridad de la señal.

Crimpadora

Es una herramienta que se utiliza para unir un conector a un cable mediante el proceso de crimpado. Es esencial para preparar cables de red y telefonía, garantizando una conexión segura y de alta calidad.

Desforradora

Esta herramienta está diseñada para retirar el aislamiento externo de los cables sin dañar los conductores internos. Es fundamental para preparar los cables antes de proceder con la conexión o el crimpado.

Tester

Un dispositivo utilizado para verificar la integridad y funcionalidad de un cable de red. Permite detectar cortocircuitos, cables cruzados, y otros problemas comunes que pueden afectar el rendimiento de la red.

Cableado horizontal

El cableado horizontal es el sistema de cables que se extiende desde los paneles de parcheo en los armarios de telecomunicaciones hasta las salidas de red en las áreas de trabajo. Es un componente crucial del cableado estructurado, ya que conecta el equipo de usuario a la red central.

Cableado vertical

también conocido como backbone o cableado de backbone, es el sistema de cables que conecta los diferentes pisos de un edificio o diferentes áreas dentro de un mismo complejo. Este tipo de cableado es crucial para establecer la columna vertebral de una red de comunicaciones, ya que interconecta los armarios de telecomunicaciones (también conocidos como MDF y IDF) ubicados en distintos pisos o zonas.

¿Qué son y para qué sirven las normas internacionales? (Ponga un ejemplo de una normativa aplicada a redes informáticas)

ormas internacionales son estándares desarrollados por organizaciones internacionales que definen criterios específicos en diversos campos para garantizar calidad, seguridad, interoperabilidad y eficiencia a nivel global. Sirven para establecer un lenguaje común entre diferentes países y empresas, facilitando el comercio y la colaboración internacional. Ejemplo: Una normativa aplicada a redes informáticas es la ISO/IEC 27001, que se refiere a la gestión de seguridad de la información. Esta norma ayuda a las organizaciones a proteger sus activos de información de manera efectiva.

Diferentes topologías, categorías y modelos de red

Topologías de red: Estrella: Todos los dispositivos están conectados a un nodo central (un switch o hub). Bus: Todos los dispositivos comparten un único cable de comunicación. Anillo: Cada dispositivo está conectado a otros dos, formando un circuito cerrado. Malla: Cada dispositivo se conecta a varios otros, proporcionando redundancia. Categorías de red: LAN (Local Area Network): Red que cubre una área limitada, como una oficina o un edificio. WAN (Wide Area Network): Red que cubre una área geográfica extensa, como un país o continente. MAN (Metropolitan Area Network): Red que cubre un área metropolitana o ciudad. Modelos de red: Modelo OSI: Un modelo conceptual con 7 capas (física, enlace de datos, red, transporte, sesión, presentación y aplicación) para entender cómo los diferentes protocolos interactúan en una red. Modelo TCP/IP: Un modelo de 4 capas (acceso a la red, Internet, transporte y aplicación) que describe cómo los datos son transmitidos a través de una red.

Normas de cableado estructurado y la funcionalidad de cada una de ellas/h3>

El cableado estructurado se refiere a un sistema de cableado que soporta múltiples tipos de tráfico, como voz, datos y video, y que se organiza de forma estandarizada para ser flexible y escalable. Normas comunes: ANSI/TIA-568: Define los estándares para el cableado de telecomunicaciones en edificios comerciales. ISO/IEC 11801: Proporciona un estándar global para el cableado estructurado, cubriendo tanto edificios comerciales como residenciales. Funcionalidad: Categoría 5e (Cat 5e): Soporta hasta 1000 Mbps (Gigabit Ethernet). Categoría 6 (Cat 6): Soporta hasta 10 Gbps en distancias más cortas. Categoría 6a (Cat 6a): Soporta 10 Gbps hasta 100 metros.
